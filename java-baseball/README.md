# 기능 명세

- 랜덤한 세자리 숫자 만들기
- 사용자에게 세자리 숫자 입력받기
- 랜덤한 숫자와 입력받은 값 비교하여 결과 출력
- 재시작 여부 입력받기
<br>

---

# 테스트케이스 명세

- 랜덤한 세자리 숫자 값이 3자리인지 검사
- 랜덤한 세자리 숫자 값이 모두 숫자인지 검사
- 사용자가 입력한 값이 3자리인지 검사
- 사용자가 입력한 값이 모두 숫자인지 검사
- 랜덤한 숫자와 입력받은 값이 같은지 비교 테스트
<br>
